# Dashboard Management System

A complete dashboard project built with **HTML** and **CSS**, featuring sections for invoices, payment methods, and transaction tables.  
The design is fully responsive across mobile, tablet, and desktop.

## ✨ Features
- Invoice management section
- Payment methods cards
- Transaction tables with scroll support
- Responsive design for mobile, tablet, and desktop
- Clean and modern UI layout

## 🛠️ Technologies Used
- **HTML5** for structure
- **CSS3** for styling
- **CSS Grid** for layout organization
- **Flexbox** for flexible alignment
- **Media Queries** for responsive design

## 📱 Responsiveness
- Mobile view: optimized with smaller font sizes and scrollable tables
- Tablet view: balanced two-column layout without empty spaces
- Desktop view: full dashboard experience with wide layout
